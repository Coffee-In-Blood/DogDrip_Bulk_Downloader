# DogDrip Attachment Bulk Downloader

DogDrip 게시글의 첨부파일 다운로드 링크를 모아서 Chrome 툴바 팝업에서 한 번에 다운로드하는 확장 프로그램입니다.

페이지 안에는 버튼이나 패널을 삽입하지 않습니다. 확장 아이콘을 눌렀을 때만 현재 탭에서 첨부파일 링크를 읽어옵니다.

## 설치

1. Chrome에서 `chrome://extensions`를 엽니다.
2. 오른쪽 위의 `개발자 모드`를 켭니다.
3. `압축해제된 확장 프로그램을 로드`를 누릅니다.
4. `다운로드받은 위치\dogdrip-bulk-downloader` 폴더를 선택합니다.

## 사용

1. DogDrip에 로그인합니다.
2. 첨부파일 버튼이 활성화되는 게시글을 엽니다.
3. Chrome 툴바의 `DogDrip 첨부 일괄 다운로드` 확장 아이콘을 클릭합니다.
4. 필요하면 `사진`, `동영상`, `사진/동영상` 옵션을 선택하고 표시되는 다운로드 위치를 확인합니다.
5. `일괄 다운로드`를 누릅니다.

확장 프로그램을 설치하거나 업데이트한 직후 이미 열려 있던 DogDrip 탭에서는 콘텐츠 스크립트가 아직 주입되지 않았을 수 있습니다. 이때는 게시글 탭을 새로고침한 뒤 툴바 아이콘을 다시 누르면 됩니다.

## 다운로드 방식

파일은 압축하지 않고 개별 파일로 다운로드합니다. 확장 프로그램이 `DogDrip/...` 같은 하위 폴더를 만들지 않고, Chrome 기본 다운로드 폴더에 바로 저장합니다.

Chrome 확장 프로그램은 보안상 기본 다운로드 폴더의 절대 경로를 읽을 수 없습니다. 팝업에는 `Chrome 기본 다운로드 폴더`로 표시하고, `다운로드 폴더 열기` 버튼으로 실제 폴더를 열 수 있게 했습니다.

Chrome 다운로드 설정에서 `다운로드 전에 각 파일의 저장 위치 확인`이 켜져 있으면 파일마다 저장 창이 뜰 수 있으니, 일괄 다운로드용으로는 해당 옵션을 꺼두는 편이 좋습니다.

## 구조

- `manifest.json`: Chrome 확장 프로그램 설정
- `icons/`: 확장 프로그램 아이콘
- `content.js`: DogDrip 페이지에서 첨부파일 링크 수집
- `background.js`: Chrome downloads API로 다운로드 실행
- `popup.html`, `popup.css`, `popup.js`: 툴바 팝업 UI

## Chrome 웹 스토어 제출 메모

이 확장 프로그램은 DogDrip 공식 제공 확장이 아닌 비공식 보조 도구입니다. 스토어 설명에는 사용자가 DogDrip 게시글 탭에서 확장 아이콘을 눌렀을 때만 첨부 이미지와 동영상 링크를 읽고, 선택한 파일을 Chrome 다운로드 목록에 추가한다고 적는 것을 권장합니다.

권한 사용 사유:

- `activeTab`: 사용자가 확장 아이콘을 누른 현재 DogDrip 탭을 확인하기 위해 사용합니다.
- `downloads`: 사용자가 선택한 DogDrip 첨부파일을 Chrome 다운로드 기능으로 저장하기 위해 사용합니다.
- `scripting`: 콘텐츠 스크립트가 아직 주입되지 않은 DogDrip 탭에서 첨부파일 링크 수집 스크립트를 다시 주입하기 위해 사용합니다.
- `https://www.dogdrip.net/*`: DogDrip 게시글의 첨부파일 링크만 읽기 위해 사용합니다.

개인정보 처리 설명:

- 로그인 정보, 쿠키, 비밀번호, 결제 정보, 개인 식별 정보를 수집하거나 외부 서버로 전송하지 않습니다.
- 현재 DogDrip 페이지에 표시된 첨부파일 링크, 파일명, 페이지 제목만 브라우저 안에서 처리합니다.
- 다운로드 요청은 Chrome의 기본 다운로드 API를 통해 사용자의 기기에서만 실행됩니다.
- 광고, 추적, 분석, 원격 코드 로딩 기능은 없습니다.
- 자세한 내용은 `https://coffee-in-blood.github.io/DogDrip_Bulk_Downloader/privacy.html`를 참고하세요

## 문제 해결

- 툴바 팝업이 계속 로딩 중이면 `chrome://extensions`에서 확장 프로그램을 새로고침하고 DogDrip 탭도 새로고침하세요.
- 현재 버전은 팝업을 열 때 콘텐츠 스크립트가 없으면 자동으로 다시 주입합니다.
- 다운로드가 시작되지 않는 것처럼 보이면 Chrome 오른쪽 위 다운로드 목록에서 차단 또는 확인 대기 항목이 있는지 확인하세요.
