# 개인정보처리방침

DogDrip Attachment Bulk Downloader는 DogDrip 게시글의 첨부 이미지와 동영상을 사용자가 쉽게 다운로드할 수 있도록 돕는 비공식 Chrome 확장 프로그램입니다.

## 1. 수집 또는 처리하는 정보

이 확장 프로그램은 사용자가 DogDrip 게시글 탭에서 확장 프로그램 아이콘을 클릭했을 때, 현재 페이지에 표시된 첨부파일 다운로드 링크, 파일명, 이미지 및 동영상 URL, 페이지 제목을 브라우저 내부에서 처리할 수 있습니다.

이 확장 프로그램은 이름, 이메일 주소, 주소, 전화번호, 비밀번호, 쿠키, 로그인 정보, 결제 정보, 건강 정보, 위치 정보와 같은 개인 식별 정보를 수집하지 않습니다.

## 2. 정보 사용 목적

처리된 정보는 사용자가 선택한 DogDrip 첨부 이미지 또는 동영상을 Chrome 다운로드 목록에 추가하기 위한 목적으로만 사용됩니다.

## 3. 정보 저장

이 확장 프로그램은 수집한 정보를 외부 서버에 저장하지 않습니다. 첨부파일 링크와 파일명은 다운로드 요청을 처리하는 동안 브라우저 내부에서만 일시적으로 사용됩니다.

## 4. 정보 공유

이 확장 프로그램은 사용자 데이터를 개발자, 제3자, 광고 네트워크, 분석 서비스 또는 외부 서버와 공유하지 않습니다.

## 5. 원격 코드 및 추적

이 확장 프로그램은 원격 코드를 로드하거나 실행하지 않습니다. 광고, 사용자 추적, 분석 기능을 포함하지 않습니다.

## 6. 권한 사용

이 확장 프로그램은 다음 권한을 사용합니다.

- `activeTab`: 사용자가 확장 프로그램 아이콘을 클릭한 현재 DogDrip 탭을 확인하기 위해 사용합니다.
- `downloads`: 사용자가 선택한 DogDrip 첨부파일을 Chrome 다운로드 기능으로 저장하기 위해 사용합니다.
- `scripting`: 콘텐츠 스크립트가 아직 주입되지 않은 DogDrip 탭에서 첨부파일 링크 수집 스크립트를 다시 주입하기 위해 사용합니다.
- `https://www.dogdrip.net/*`: DogDrip 게시글의 첨부파일 링크만 읽기 위해 사용합니다.

## 7. 문의

개인정보처리방침과 관련된 문의는 Chrome 웹 스토어에 등록된 게시자 연락처 이메일로 연락할 수 있습니다.
